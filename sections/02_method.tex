\section{Method}
\label{sec:method}
Inline math such as $\mathbf{y} = \mathbf{W}\mathbf{x} + \mathbf{b}$ uses
Charter-keyed glyphs so it blends with the body. Display math:
\begin{equation}
  \label{eq:softmax}
  \mathrm{softmax}(\mathbf{z})_i = \frac{e^{z_i}}{\sum_{j=1}^{K} e^{z_j}},
  \qquad i = 1,\dots,K.
\end{equation}
